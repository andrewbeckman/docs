---
title: "GetReportBuilderConfig"
api: "POST /GetReportBuilderConfig"
description: "Retrieve the available report configuration options including base data types, columns, filters, and default parameters."
---

## Overview

`GetReportBuilderConfig` returns the full configuration schema for the report builder. This endpoint provides metadata about all available report types, their supported columns, filters, and default parameters. Use this before calling `GetReportBuilder` to understand what options are available for building reports.

## Request

This endpoint takes no parameters.

<RequestExample>
```json
{}
```
</RequestExample>

## Response

<ResponseField name="mappings_by_base_data" type="map<int32, Mapping>">
  A map from `BaseData` enum values to their corresponding configuration mappings.
  
  <Expandable title="Mapping properties">
    <ResponseField name="base_data" type="BaseData">
      The base data type enum (e.g., `SOLD_PRODUCTS`, `CUSTOMERS`, `TRANSACTIONS`).
    </ResponseField>
    <ResponseField name="all_columns" type="GroupV2[]">
      All available column types for this base data.
    </ResponseField>
    <ResponseField name="filters" type="Filter.Attribute[]">
      Available filter attributes (e.g., `LOCATION_ID`, `CUSTOMER_ID`, `PRODUCT_ID`).
    </ResponseField>
    <ResponseField name="label_columns" type="GroupV2[]">
      Columns that can be used as grouping labels (enum values < 10000).
    </ResponseField>
    <ResponseField name="metric_columns" type="GroupV2[]">
      Columns that represent numeric metrics (enum values >= 10000).
    </ResponseField>
    <ResponseField name="default_params" type="ReportParamsV2">
      The default report parameters for this base data type.
    </ResponseField>
    <ResponseField name="is_single_date" type="boolean">
      Whether this report type uses a single date rather than a date range.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="group_names" type="map<int32, string>">
  A map from `GroupV2` enum values to their human-readable display names.
</ResponseField>

<ResponseField name="base_data_names" type="map<int32, string>">
  A map from `BaseData` enum values to their human-readable display names.
</ResponseField>
